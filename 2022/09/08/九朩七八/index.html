Hexo + Github 搭建个人主页的小细节

  • github 新建仓库时，仓库名要命名为your_name.github.io 才能将博客的域名设置为your_name.github.io，否则还需要在 .io 后面加上仓库的名字。

  • 在 hexo/_config.yml 中找到并设置post_asset_folder: true，这样在hexo new page xxxx 创建文章时会同时创建与文章名用于保存相应附件（图片等），以实现使用相对地址在网页中引入图片。
